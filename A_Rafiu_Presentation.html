AES & BLOCK CIPHERS
INTERMEDIATE
AES MODES — KNOW YOUR TARGET
MODEVULNERABILITY
ECBSAME BLOCK = SAME CT ⚠
CBCBit-flip, padding oracle
CTRNonce reuse → XOR leak
GCMNonce reuse → key recovery
OFBNonce reuse → keystream
⚡ ECB PENGUIN ECB encrypts each 16-byte block independently. Identical plaintext blocks produce identical ciphertext — patterns leak!
ECB BYTE-AT-A-TIME
  • Pad input so unknown secret byte is last in block
  • Brute-force all 256 values for that position
  • Match against oracle output to reveal byte
  • Repeat for each byte of secret
CBC PADDING ORACLE
# 1. Have encryption oracle + padding check # 2. Flip bits in C[i-1] to find valid padding # 3. P[i][j] = 0x01 XOR modified_C[i-1][j] # 4. Repeat for full block → full plaintext from pwntools import * # pwntools has padding oracle solver! pwnlib.util.iters.bruteforce()
CTR NONCE REUSE — CRITICAL BUG
# Two messages, same key+nonce: C1 = P1 XOR keystream C2 = P2 XOR keystream # XOR ciphertexts → XOR plaintexts C1 XOR C2 = P1 XOR P2 # If you know P1 → recover P2! # Known-plaintext: submit your own P1

XOR & STREAM CIPHERS
INTERMEDIATE
XOR PROPERTIES (MEMORIZE)
A XOR A=0
A XOR 0=A
C XOR K=P
C XOR P=K← KEY RECOVERY!
C1 XOR C2=P1 XOR P2
🔑 CRIB DRAG If you know part of the plaintext (like "flag{"), XOR it with ciphertext at each offset. When you get readable output, you've found the keystream portion.
LFSR ATTACKS
  • LFSR output is periodic and predictable from sufficient output bits.
  • Berlekamp-Massey algorithm recovers feedback polynomial from ~2n bits.
  • Tool: sympy or implement B-M manually in Python.
SINGLE-BYTE XOR — FULL SOLVER
def score(text): freq = "etaoinshrdlu" return sum(1 for c in text.lower() if c in freq) def crack_xor(ciphertext): best = (0, 0, b'') for key in range(256): pt = bytes(b^key for b in ciphertext) s = score(pt) if s > best[0]: best = (s, key, pt) return best
REPEATING KEY XOR
# 1. Find key length: IC or Hamming dist # 2. Split into key_len slices # 3. Solve each slice as single-byte XOR # 4. Combine key bytes def hamming(a, b): return sum(bin(x^y).count('1') for x,y in zip(a,b)) # Lowest normalized dist = key length
